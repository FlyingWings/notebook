### 全局变量
- 定义：在脚本运行前就定义的符号表中的变量，可全局调用
- 全局变量$GLOBALS的初始化
    - `php_request_startup()` 请求初始化
    - 注册符号表，故而每次请求的$GLOBALS都是不同的
- $_GET, $_POST等变量
    - 同上，请求初始化
    - `php_hash_environment()` 初始化这些变量
### 静态变量
- 定义：静态分配（不会重复初始化）
- 分类：
    - 静态全局变量：在初始化阶段就定义，不会重复定义，除非`unset()`释放，否则一直存在
    - 静态局部变量：函数内部变量，该函数在声明时初始化，并且不会被销毁
    - 静态成员变量：类静态变量
